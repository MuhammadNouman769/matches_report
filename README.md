# Matches Report

This is the Matches Report project built with Django and Django REST Framework. 
It manages match data, generates reports, and provides API endpoints for CRUD operations.

## Features
- REST API endpoints for managing matches
- Browsable API using DRF
- Django admin panel integration

## Installation
1. Clone the repo
2. Create and activate virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Run migrations: `python manage.py migrate`
5. Run server: `python manage.py runserver`
